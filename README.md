วิธีใช้งาน
1.ใส่ตัวเลขที่ช่องใส่จำนวนคน
2.กด confirm
3.ใส่ชื่อคนที่ต้องการที่ช่องใส่ชื่อคน
4.กด send
5.ทำข้อ 3-4 จนกว่าจะใส่ชื่อครยตามจำนวน
6.กดปุ่มคนสีฟ้าเพื่อนำคนเข้าไปเรียงในแต่ละบ้าน
