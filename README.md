วิธีใช้งาน<br />
1.ใส่ตัวเลขที่ช่องใส่จำนวนคน<br />
2.กด confirm<br />
3.ใส่ชื่อคนที่ต้องการที่ช่องใส่ชื่อคน<br />
4.กด send<br />
5.ทำข้อ 3-4 จนกว่าจะใส่ชื่อครบตามจำนวน<br />
6.กดปุ่มคนสีฟ้าเพื่อนำคนเข้าไปเรียงในแต่ละบ้าน<br />
